# Система одометрии мобильного робота на основе датчиков оптического потока
### Выпускная квалификационная работа студента группы 233 Савина Никиты.

Два оптических датчика ADNS3080, наблюдающих поверхность под левым и правым
бортом мобильной платформы, отправляют данные о смещении, а микроконтроллер
STM32F407 упаковывает эти значения в UDP-датаграмму и отправляет на компьютер
по Ethernet. 

Для реализации системы использована библиотека libopencm3, операционная 
система FreeRTOS и стек LwIP для работы с Ethernet.
